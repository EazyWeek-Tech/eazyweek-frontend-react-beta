# Insight Appointment Booking Service

A modern web application built with **React + Vite**.  
This project includes modular components, backend integration via API handlers, and is ready for deployment.

## Project Overview

- **Frontend:** React 18 + Vite
- **Language:** TypeScript / JavaScript
- **Styling:** TailwindCSS / CSS Modules / Styled Components
- **Backend:** Connects to HTTP Handlers in ASP Web Form Project
- **Deployment:** Azure server

## Setup Instructions
# 1. Clone the repo
git clone https://github.com/nasreenqrsh/aptmodule.git
cd aptmodule

# 2. Install dependencies
npm install
npm install gh-pages --save-dev

# 3. Run the app locally
npm run dev

## Build for Production
npm run build

## Preview Production Build
npm run preview

## Unit Testing
npm run test

## Backend Handlers(/CustomerHandler)
Service handles call to create customer using fetch with error handling.
